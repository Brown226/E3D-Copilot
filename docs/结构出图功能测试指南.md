## 支持的投影方向

| 方向 | 说明 |
|------|------|
| `top` | 俯视图（平面图） |
| `front` / `south` | 正视图 / 南立面 |
| `back` / `north` | 后视图 / 北立面 |
| `left` / `west` | 左视图 / 西立面 |
| `right` / `east` | 右视图 / 东立面 |

---

## 支持的元素类型

| 类型 | 说明 | 提取属性 |
|------|------|----------|
| SCTN | 梁/柱 | StartPosition, EndPosition, Width, Depth |
| STWL | 墙 | WorldPosition + 边界点估算 |
| FRMW | 框架 | StartPosition, EndPosition |
| GENSEC | 通用截面 | 同 SCTN |
| STRU | 结构容器 | WorldPosition + 子元素包围盒 |

---

## 输出文件说明

生成的 DXF 文件包含以下图层：

| 图层名 | 内容 | 颜色 |
|--------|------|------|
| STRUCTURE | 结构实线 | 白色(7) |
| STRUCTURE-HIDDEN | 结构隐藏线 | 灰色(8) |
| CENTERLINE | 中心线 | 红色(1) |
| DIMENSION | 尺寸标注 | 绿色(3) |
| TEXT | 文字 | 白色(7) |
| TITLE-BLOCK | 标题栏 | 白色(7) |
| FRAME | 图框 | 白色(7) |

## 常见问题

### Q1: 元素提取失败怎么办？
A: 系统会自动回退到模拟数据。如需真实数据，请确保：
- E3D 正在运行
- 元素名称正确
- 当前用户有权限访问元素

### Q2: DXF 文件打不开？
A: 检查：
- 文件路径是否正确
- 磁盘空间是否充足
- AutoCAD 版本是否支持

### Q3: 消隐效果不理想？
A: 消隐基于深度排序，复杂遮挡可能需要手动调整

### Q4: 如何调整图框大小？
A: 系统会根据内容自动选择 A3 或 A2 图框
