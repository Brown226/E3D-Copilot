## 四、命名规范

| 项 | 规范 | 示例 |
|----|------|------|
| 项目名称 | `E3DCopilot.{Area}` | `E3DCopilot.Core` |
| 命名空间 | `E3DCopilot.{Area}.{Sub}` | `E3DCopilot.Tools.Query` |
| 类名 | PascalCase | `AgentLoop` |
| 接口 | `I` 前缀 | `ICopilotProvider` |
| 文件命名 | 与类名一致 | `AgentLoop.cs` |
| 测试文件 | `{ClassName}Tests.cs` | `AgentLoopTests.cs` |

## 五、依赖关系

```
E3DCopilot.UI (WinForms)
     ↓
E3DCopilot.Tools (PML + E3D API)
     ↓
E3DCopilot.Core (纯逻辑)
     ↓
E3DCopilot.Addin (E3D 入口)
```

禁止反向依赖（Core → Tools → UI）。
