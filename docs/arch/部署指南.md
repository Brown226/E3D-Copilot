## 二、部署结构（E3D 加载方式）

```
E3D 安装目录/
├── PMLLIB/                        ← PML 索引目录
│   ├── site/                      ← 站点级别
│   │   ├── pml.index              ← PML 索引文件
│   │   └── e3dcopilot/            ← E小智 PML 工具
│   │       ├── init.pmlfnc        ← PML 加载入口
│   │       └── *.pmlfnc           ← PML 辅助函数
│   └── project/                   ← 项目级别
│       └── e3dcopilot/
│           └── custom.pmlfnc      ← 用户自定义
│
├── DotNet/                        ← .NET 程序集目录
│   ├── E3DCopilot.Addin.dll       ← Addin 入口
│   ├── E3DCopilot.Core.dll        ← 核心库
│   ├── E3DCopilot.Tools.dll       ← 工具库
│   ├── E3DCopilot.WebHost.dll     ← WebView2 宿主
│   ├── *.dll                      ← 依赖 DLL（Newtonsoft.Json 等）
│   │
│   ├── CNPE.ISO.E3D.Core.dll      ← 🆕 ISO出图 E3D 数据提取
│   ├── CNPE.ISO.Model.dll         ← 🆕 ISO出图数据模型
│   ├── CNPE.ISO.CAD.Core.dll      ← 🆕 ISO出图 CAD 生成
│   ├── Aspose.Cells.dll           ← 🆕 Excel 处理
│   └── ...
│
└── AppData/                       ← 运行时数据
    └── .e3dcopilot/               ← 自动创建（首次启动）
        ├── config.json            ← 全局配置（管理员维护）
        ├── user.json              ← 用户配置（%LOCALAPPDATA%）
        ├── user_profile.json      ← 用户画像
        ├── sessions/              ← 会话历史
        └── memory/                ← 长期记忆
```

### ISO出图相关文件说明 🆕

| 文件 | 来源 | 说明 |
|------|------|------|
| `CNPE.ISO.E3D.Core.dll` | CNPE.IC.ISO 项目 | E3D 数据提取（PipeReader/BranReader） |
| `CNPE.ISO.Model.dll` | CNPE.IC.ISO 项目 | 数据模型（PipeLineInfoModel 等） |
| `CNPE.ISO.CAD.Core.dll` | CNPE.IC.ISO 项目 | CAD 图纸生成 |
| `Aspose.Cells.dll` | CNPE.IC.ISO 项目 | Excel 文件处理（材料数据） |

**材料数据文件**（运行时读取）：
- 位置：`E小智-v1.0-开发中/lib/iso/1907/`
- 文件：`PIPE.csv`、`BOLT.csv`、`SCTN.csv`、`SUPP.csv`
- 说明：1907项目材料编码数据，由 `MaterialQueryHandler` 读取

### 部署方式说明

E3D 通过 **DotNet 目录**加载 .NET Addin DLL：

1. `E3DCopilot.Addin.dll` 放在 `E3D安装目录/DotNet/`
2. E3D 启动时通过 `Aveva.ApplicationFramework.AddinManager` 自动发现并加载
3. 从 `AddinBoot.cs` 的 `Start()` 方法启动整个 E小智

> ✅ 不需要注册表、不需要 GAC、不需要修改 E3D 配置

## 三、构建脚本

### 3.1 开发构建

```bat
:: 构建命令（cmd.bat 或 build.cmd）
msbuild src\E3DCopilot.sln /p:Configuration=Debug

:: 构建产物
:: src\E3DCopilot.Addin\bin\Debug\E3DCopilot.Addin.dll
:: src\E3DCopilot.Core\bin\Debug\E3DCopilot.Core.dll
:: src\E3DCopilot.Tools\bin\Debug\E3DCopilot.Tools.dll
:: src\E3DCopilot.UI\bin\Debug\E3DCopilot.UI.dll
```

### 3.2 发布构建

```bat
:: 发布构建
msbuild src\E3DCopilot.sln /p:Configuration=Release

:: 只输出必需的 DLL（剔除 PDB 等调试文件）
xcopy src\E3DCopilot.Addin\bin\Release\*.dll dist\DotNet\ /Y
xcopy src\E3DCopilot.Core\bin\Release\*.dll dist\DotNet\ /Y
xcopy src\E3DCopilot.Tools\bin\Release\*.dll dist\DotNet\ /Y
xcopy src\E3DCopilot.WebHost\bin\Release\*.dll dist\DotNet\ /Y

:: 🆕 复制 ISO 出图相关 DLL
xcopy lib\iso\*.dll dist\DotNet\ /Y

:: 复制 PML 工具
xcopy pml\*.pmlfnc dist\PMLLIB\site\e3dcopilot\ /Y

:: 复制全局配置文件
copy config\default.json dist\config.json
```

### 3.3 一键部署

```bat
:: deploy.cmd — 一键部署到目标 E3D 安装目录
set E3D_DIR=C:\Aveva\E3D\1.1

:: 停止 E3D（如果有运行）
taskkill /f /im E3D.exe 2>nul

:: 复制 DLL
xcopy dist\DotNet\* %E3D_DIR%\DotNet\ /Y

:: 复制 PML
xcopy dist\PMLLIB\site\e3dcopilot\* %E3D_DIR%\PMLLIB\site\e3dcopilot\ /Y

:: 复制配置
mkdir %E3D_DIR%\AppData\.e3dcopilot 2>nul
copy dist\config.json %E3D_DIR%\AppData\.e3dcopilot\config.json

echo 部署完成。请重启 E3D。
```

## 四、PML 索引注册

### 4.1 PML 索引文件

`pml.index` 文件告诉 E3D 哪些 PML 函数可用：

```pml
-- pml.index（E3D 自动发现）
-- 无需手动注册，E3D 扫描 PMLLIB 目录自动建立索引

-- E小智 PML 辅助函数入口
e3dcopilot/init.pmlfnc
```

### 4.2 PML 初始化脚本

`init.pmlfnc` 在 E3D 启动时加载：

```pml
!function e3dcopilot_init()
    handle (any)
        $p E小智 PML 工具已加载
    endhandle
endfunction

e3dcopilot_init()
```

## 八、升级与迁移

### 8.1 升级流程

```bat
:: 1. 停止 E3D
taskkill /f /im E3D.exe

:: 2. 备份旧版本
move %E3D_DIR%\DotNet\E3DCopilot.* dist\backup\

:: 3. 删除旧版本
del %E3D_DIR%\DotNet\E3DCopilot.* 2>nul
del %E3D_DIR%\DotNet\Newtonsoft.Json.dll 2>nul

:: 4. 部署新版本
xcopy dist\DotNet\* %E3D_DIR%\DotNet\ /Y

:: 5. 重启 E3D
```

### 8.2 数据迁移

- `user_profile.json`：向前兼容，旧版本升级不会丢失习惯数据
- `sessions/`：JSONL 格式，版本无关
- `memory/`：SQLite FTS5，schema 变更时自动迁移

## 九、故障排除

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| E3D 启动时无 E小智 菜单 | DLL 未正确部署 | 检查 DotNet 目录是否包含 E3DCopilot.Addin.dll |
| Panel 打不开 | Addin.Start() 异常 | 查看 E3D 日志 / AppData 下的 .e3dcopilot 日志 |
| LLM 连接失败 | vLLM 未启动或端口不对 | 检查 config.json 中的 base_url |
| PML 执行失败 | PML 脚本语法错误 | 检查 execute_pml 的日志输出 |
| 配置丢失 | config.json 被覆盖 | 从 AppData 目录恢复备份 |
