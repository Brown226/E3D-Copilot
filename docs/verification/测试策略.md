# E小智 v1.0 — 测试策略

> 在没有 E3D 完整环境时如何测试，以及有环境时的验证流程。

---

## 一、测试分层

```
┌─────────────────────────────────────────┐
│  L1: 单元测试（无需 E3D 环境）           │  ← 最多，最快
├─────────────────────────────────────────┤
│  L2: 集成测试（Mock E3D API）           │  ← 中等
├─────────────────────────────────────────┤
│  L3: E3D 环境测试（真实环境）           │  ← 最少，最慢
└─────────────────────────────────────────┘
```

## 二、L1 单元测试（无需 E3D）

### 2.1 可测试的模块

| 模块 | 测试内容 | 依赖 |
|------|---------|------|
| AgentLoop | 循环逻辑、流式处理 | Mock Provider + Mock Tool |
| CopilotController | 命令路由、状态管理 | Mock Agent |
| ToolRegistry | 注册、查找、Schema 生成 | 无 |
| ToolPolicy | 权限判断、预设加载 | 无 |
| PmlGenerator | PML 代码生成 | 无 |
| SanitizeToolPairing | 消息修复逻辑 | 无 |
| EventSystem | 事件分发 | 无 |

### 2.2 单元测试示例

```csharp
[TestClass]
public class ToolRegistryTests
{
    [TestMethod]
    public void Register_Tool_ShouldBeFindable()
    {
        var registry = new ToolRegistry();
        var tool = new MockTool();
        
        registry.Register(tool);
        
        Assert.IsNotNull(registry.Get("mock_tool"));
    }
    
    [TestMethod]
    public void Get_UnknownTool_ShouldReturnNull()
    {
        var registry = new ToolRegistry();
        
        Assert.IsNull(registry.Get("nonexistent"));
    }
    
    [TestMethod]
    public void Schemas_ShouldReturnAllRegisteredTools()
    {
        var registry = new ToolRegistry();
        registry.Register(new MockTool1());
        registry.Register(new MockTool2());
        
        var schemas = registry.Schemas();
        
        Assert.AreEqual(2, schemas.Count);
    }
}

[TestClass]
public class PmlGeneratorTests
{
    [TestMethod]
    public void GenerateQuery_ShouldProduceValidPml()
    {
        var generator = new PmlGenerator();
        
        var pml = generator.GenerateQuery("PIPE", "*DN100*", "ZONE-01");
        
        Assert.IsTrue(pml.Contains("coll all PIPE"));
        Assert.IsTrue(pml.Contains("Matchwild(name,'*DN100*')"));
        Assert.IsTrue(pml.Contains("for $!ZONE-01"));
    }
    
    [TestMethod]
    public void GenerateBatchSet_ShouldProduceValidPml()
    {
        var generator = new PmlGenerator();
        
        var pml = generator.GenerateBatchSet("PIPE", "WTHK", "SCH40", "*DN100*", "ZONE-01");
        
        Assert.IsTrue(pml.Contains("coll all PIPE"));
        Assert.IsTrue(pml.Contains(":WTHK = 'SCH40'"));
    }
}

[TestClass]
public class ToolPolicyTests
{
    [TestMethod]
    public void GetMode_UnknownTool_ShouldDefaultToAsk()
    {

        Assert.AreEqual(ApprovalMode.Ask, policy.GetMode("unknown_tool"));

        policy.Set("query_elements", ApprovalMode.Auto);
        
        Assert.AreEqual(ApprovalMode.Auto, policy.GetMode("query_elements"));
    }
```

## 三、L2 集成测试（Mock E3D）

### 3.1 Mock E3D API

```csharp
/// <summary>
/// Mock E3D Database，用于集成测试
/// </summary>
public class MockE3DDatabase : IE3DDatabase
{
    private Dictionary<string, MockElement> elements = new Dictionary<string, MockElement>();
    
    public MockE3DDatabase()
    {
        // 预置测试数据
        AddElement("PIPE-001", "PIPE", new Dictionary<string, string>
        {
            {"DIA", "100"}, {"WTHK", "SCH30"}, {"SPEC", "CS300"}
        });
        AddElement("PIPE-002", "PIPE", new Dictionary<string, string>
        {
            {"DIA", "100"}, {"WTHK", "SCH30"}, {"SPEC", "CS300"}
        });
        AddElement("EQUI-001", "EQUI", new Dictionary<string, string>
        {
            {"ROOM_NO", "RM-101"}
        });
    }
    
    public DbElement GetElement(string name)
    {
        return elements.TryGetValue(name, out var elem) ? elem : null;
    }
    
    public List<DbElement> Query(string type, string pattern, string scope)
    {
        return elements.Values
            .Where(e => e.Type == type)
            .Where(e => pattern == null || MatchWildcard(e.Name, pattern))
            .ToList();
    }
    
    public void AddElement(string name, string type, Dictionary<string, string> attrs)
    {
        elements[name] = new MockElement(name, type, attrs);
    }
}
```

### 3.2 集成测试示例

```csharp
[TestClass]
public class QueryElementsToolTests
{
    private MockE3DDatabase database;
    private QueryElementsTool tool;
    
    [TestInitialize]
    public void Setup()
    {
        database = new MockE3DDatabase();
        tool = new QueryElementsTool(database);
    }
    
    [TestMethod]
    public async Task Execute_SimpleQuery_ShouldReturnElements()
    {
        var args = new QueryArgs { Type = "PIPE", Pattern = "*DN100*" };
        
        var result = await tool.ExecuteAsync(JsonSerializer.Serialize(args));
        
        Assert.IsTrue(result.Contains("PIPE-001"));
        Assert.IsTrue(result.Contains("PIPE-002"));
    }
    
    [TestMethod]
    public async Task Execute_NoMatch_ShouldReturnEmpty()
    {
        var args = new QueryArgs { Type = "PIPE", Pattern = "*DN200*" };
        
        var result = await tool.ExecuteAsync(JsonSerializer.Serialize(args));
        
        Assert.IsTrue(result.Contains("\"count\":0"));
    }
}
```

## 四、L3 E3D 环境测试

### 4.1 测试环境要求

- E3D 安装完成
- 测试模型已加载
- vLLM + Qwen 服务运行中
- 插件已安装

### 4.2 手动测试清单

#### 基础功能

- [ ] 插件加载成功（E3D 启动无报错）
- [ ] Copilot 窗口可正常打开
- [ ] 可输入文本并发送
- [ ] AI 能正常回复

#### 工具调用

- [ ] query_elements 能正确查询元素
- [ ] get_attributes 能正确读取属性
- [ ] set_attribute 能正确修改属性（需确认）
- [ ] check_exists 能正确检查存在性
- [ ] get_children 能正确获取子元素
- [ ] navigate_to 能正确跳转

#### PML 执行

- [ ] PML 脚本能正常执行
- [ ] PML 输出能正确解析
- [ ] PML 错误能正确处理

#### 审批流程

- [ ] Plan Mode 下写工具被阻止
- [ ] 审批弹窗正常显示
- [ ] 键盘快捷键（1/2/3/4）正常工作
- [ ] 批量操作前显示计划

#### 边界场景

- [ ] vLLM 断连时有友好提示
- [ ] 不存在的元素查询返回空结果
- [ ] 无效 PML 脚本有错误提示
- [ ] 大量数据查询不卡顿

## 六、持续集成

### 6.1 CI 流程

```
代码提交
  ↓
L1 单元测试（自动）
  ↓ 通过
L2 集成测试（自动）
  ↓ 通过
构建 DLL
  ↓
手动 L3 测试（有 E3D 环境时）
```

### 6.2 测试覆盖率目标

| 模块 | 目标覆盖率 | 说明 |
|------|:---------:|------|
| ToolRegistry | 90% | 核心注册逻辑 |
| ToolPolicy | 90% | 权限判断 |
| PmlGenerator | 80% | PML 生成 |
| AgentLoop | 70% | 核心循环 |
