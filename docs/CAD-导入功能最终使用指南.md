   - 元素数: 2

### **② AutoCAD 软件联动** ✅ 已实现

**使用场景：** 从运行中的 AutoCAD 直接获取选中的图形

**使用步骤：**

#### **步骤 1：启动 AutoCAD**
- 打开 AutoCAD 并加载图纸
- 在图纸中选择要导入的图形（线段、多段线等）

#### **步骤 2：连接 AutoCAD**
```
用户：连接 AutoCAD

AI：好的，我来连接 AutoCAD。

[自动调用 import_cad 工具，action=connect_autocad]

AI：✅ 已成功连接到 AutoCAD
   - 当前图纸: 建筑平面图.dwg
   
   现在可以使用以下操作：
   - get_selected: 获取选中的对象
   - import_from_autocad: 从 AutoCAD 导入选中的对象
```

#### **步骤 3：获取选中对象**
```
用户：获取 AutoCAD 中选中的对象

AI：好的，我来获取您选中的对象。

[自动调用 import_cad 工具，action=get_selected]

AI：✅ 获取到 5 个对象
   - 图纸: 建筑平面图.dwg
   - 线段数: 8
   
   线段列表:
   [1] (0, 0, 0) → (5000, 0, 0) 长度: 5000mm
   [2] (5000, 0, 0) → (5000, 3000, 0) 长度: 3000mm
   ...
```

#### **步骤 4：导入到 E3D**
```
用户：把这些对象导入到 E3D，墙高3米，墙厚200mm

AI：好的，我来导入选中的对象。

[自动调用 import_cad 工具，action=import_from_autocad]

AI：✅ 从 AutoCAD 导入准备完成
   - 图纸: 建筑平面图.dwg
   - 原始对象: 5 个
   - 有效线段: 8 条
   - 将创建元素: 8 个
   - 墙高: 3000mm
   - 墙厚: 200mm
   
   已生成 PML 脚本，是否需要我在 E3D 中执行？
```

## 🔧 **工具调用格式**

### **1. 连接 AutoCAD**
```json
{
  "action": "connect_autocad"
}
```

### **2. 获取选中对象**
```json
{
  "action": "get_selected"
}
```

### **3. 从 AutoCAD 导入**
```json
{
  "action": "import_from_autocad",
  "wall_height": 3000,
  "wall_thickness": 200,
  "layer_filter": ["墙体", "WALL"]
}
```

### **4. 坐标字符串导入**
```json
{
  "action": "import",
  "paths_string": "[(0,0,0),(5000,0,0)],[(5000,0,0),(5000,3000,0)]",
  "wall_height": 3000,
  "wall_thickness": 200
}
```

### **5. 预览导入结果**
```json
{
  "action": "preview",
  "paths_string": "[(0,0,0),(5000,0,0)]"
}
```

### **6. 生成 PML 脚本**
```json
{
  "action": "generate_pml",
  "paths_string": "[(0,0,0),(5000,0,0)]",
  "site_name": "MY_SITE",
  "zone_name": "MY_ZONE"
}
```

## ⚠️ **注意事项**

### **AutoCAD 联动要求**
1. AutoCAD 必须正在运行
2. 必须打开至少一个图纸文件
3. 需要在 AutoCAD 中选择要导入的图形

### **坐标格式**
- 单位：毫米（mm）
- 格式：`[(x1,y1,z1),(x2,y2,z2)]`
- Z 坐标可选，默认为 0

### **支持的 AutoCAD 实体类型**
- ✅ 直线（Line）
- ✅ 多段线（Polyline）
- ✅ 圆弧（Arc）- 自动离散化为线段
- ✅ 圆（Circle）- 自动离散化为线段
- ✅ 块参照（BlockReference）
