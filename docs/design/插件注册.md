# E小智 v1.0 — E3D Addin 注册机制

> 如何将 E小智 插件注册到 E3D 环境中运行。

---

## 一、E3D 插件架构

```
E3D 启动
  ↓
加载 Addin 配置文件
  ↓
实例化 Addin 类
  ↓
调用 Start() 方法
  ↓
注册 DockedWindow / CommandBar / ButtonTool
  ↓
插件运行
```

---

## 二、Addin 入口类

```csharp
using Aveva.ApplicationFramework;
using Aveva.ApplicationFramework.Presentation;

namespace E3DCopilot
{
    /// <summary>
    /// E小智 E3D AI Copilot 插件入口
    /// 继承 Aveva.ApplicationFramework.Addin
    /// </summary>
    public class CopilotAddin : Addin
    {
        private DockedWindow dockedWindow;
        private CopilotForm copilotForm;
        private CopilotController controller;

        /// <summary>
        /// 插件名称
        /// </summary>
        public override string Name
        {
            get { return "E3DCopilot"; }
        }

        /// <summary>
        /// 插件描述
        /// </summary>
        public override string Description
        {
            get { return "E3D AI Copilot - 自然语言驱动的 E3D 操作助手"; }
        }

        /// <summary>
        /// 插件启动时调用
        /// </summary>
        public override void Start()
        {
            try
            {
                // 1. 创建 Controller（核心逻辑层）
                controller = new CopilotController();

                // 2. 创建 UI 窗口
                copilotForm = new CopilotForm(controller);

                // 3. 注册到 E3D DockedWindow（4 个参数，含 DockedPosition）
                dockedWindow = WindowManager.Instance.CreateDockedWindow(
                    "E3DCopilot",               // 窗口 Key
                    "E小智 Copilot",             // 窗口标题
                    copilotForm,                // UserControl
                    DockedPosition.Right         // 停靠位置（Right/Left/Top/Bottom/Floating/Docked）
                );

                // 4. 注册工具栏按钮（可选）
                RegisterToolbarButton();

                // 5. 订阅 E3D 事件
                SubscribeE3DEvents();

                // 输出启动成功消息（Messaging 静态日志类不存在，用 PML 输出到消息窗）
                Command.CreateCommand("$p E小智 Copilot 已启动").RunInPdms();
            }
            catch (Exception ex)
            {
                Command.CreateCommand("$p E小智 启动失败: " + ex.Message).RunInPdms();
            }
        }

        /// <summary>
        /// 插件停止时调用
        /// </summary>
        public override void Stop()
        {
            try
            {
                // 1. 释放 Controller
                if (controller != null)
                {
                    controller.Dispose();
                    controller = null;
                }

                // 2. 关闭窗口
                if (dockedWindow != null)
                {
                    dockedWindow.Close();
                    dockedWindow = null;
                }

                // 3. 取消事件订阅
                UnsubscribeE3DEvents();

                Command.CreateCommand("$p E小智 Copilot 已停止").RunInPdms();
            }
            catch (Exception ex)
            {
                Command.CreateCommand("$p E小智 停止异常: " + ex.Message).RunInPdms();
            }
        }

        /// <summary>
        /// 注册工具栏按钮
        /// </summary>
        private void RegisterToolbarButton()
        {
            // 创建工具栏按钮
            var button = new ButtonTool(
                "E3DCopilotButton",        // 按钮 ID
                "E小智",                    // 按钮文本
                "E3D AI Copilot"           // 工具提示
            );

            // 绑定点击事件
            button.ToolClick += (sender, args) =>
            {
                if (dockedWindow != null)
                {
                    dockedWindow.Show();
                }
            };

            // 注册到 E3D 工具栏
            // 具体注册方式取决于 E3D 版本
        }

        /// <summary>
        /// 订阅 E3D 事件
        /// </summary>
        private void SubscribeE3DEvents()
        {
            // 订阅当前元素变化事件
            // 当用户在 E3D 中选择不同元素时触发
            // 可用于实现 @提及 功能
        }

        /// <summary>
        /// 取消事件订阅
        /// </summary>
        private void UnsubscribeE3DEvents()
        {
            // 取消所有事件订阅
        }
    }
}
```

## 三、Addin 配置文件

E3D 通过配置文件加载插件。配置文件通常位于 E3D 安装目录或用户目录下。

### 配置文件格式（INI 风格）

```ini
[Addin]
Name=E3DCopilot
Description=E3D AI Copilot - 自然语言驱动的 E3D 操作助手
Assembly=E3DCopilot.dll
Class=E3DCopilot.CopilotAddin
Version=1.0.0
Author=E3D-E小智 Team
Enabled=1
LoadOrder=100
```

### 或 XML 格式

```xml
<?xml version="1.0" encoding="utf-8"?>
<Addins>
  <Addin>
    <Name>E3DCopilot</Name>
    <Description>E3D AI Copilot - 自然语言驱动的 E3D 操作助手</Description>
    <Assembly>E3DCopilot.dll</Assembly>
    <Class>E3DCopilot.CopilotAddin</Class>
    <Version>1.0.0</Version>
    <Author>E3D-E小智 Team</Author>
    <Enabled>true</Enabled>
    <LoadOrder>100</LoadOrder>
  </Addin>
</Addins>
```

## 五、项目配置 (.csproj)

> E小智 有 4 个独立项目，每个项目的 csproj 配置不同。详情见 [源码结构.md](../arch/源码结构.md) 和 [部署指南.md](../arch/部署指南.md)。

### 5.1 Addin 项目（插件入口）

```xml
<Project Sdk="Microsoft.NET.Sdk">
  <PropertyGroup>
    <TargetFramework>net48</TargetFramework>
    <RootNamespace>E3DCopilot.Addin</RootNamespace>
    <AssemblyName>E3DCopilot.Addin</AssemblyName>
    <OutputType>Library</OutputType>
    <LangVersion>7.3</LangVersion>
  </PropertyGroup>

  <ItemGroup>
    <ProjectReference Include="..\E3DCopilot.Core\E3DCopilot.Core.csproj" />
    <ProjectReference Include="..\E3DCopilot.UI\E3DCopilot.UI.csproj" />
    <!-- E3D SDK 引用 -->
    <Reference Include="Aveva.ApplicationFramework">
      <HintPath>$(E3DPath)\Aveva.ApplicationFramework.dll</HintPath>
    </Reference>
    <Reference Include="Aveva.ApplicationFramework.Presentation">
      <HintPath>$(E3DPath)\Aveva.ApplicationFramework.Presentation.dll</HintPath>
    </Reference>
    <Reference Include="Aveva.Pdms.Utilities">
      <HintPath>$(E3DPath)\Aveva.Pdms.Utilities.dll</HintPath>
    </Reference>
    <Reference Include="System.Windows.Forms" />
  </ItemGroup>
</Project>
```

## 六、安装部署

> ⚠️ E3D .NET Addin 通过 **DotNet 目录**加载，不同于旧版 PDMS 的 Addins 目录。详情见 [部署指南.md](../arch/部署指南.md)。

### 步骤 1：编译插件

```bash
msbuild src\E3DCopilot.sln /p:Configuration=Release
```

### 步骤 2：复制文件

将以下文件复制到 E3D 的 DotNet 目录：

```
E3D安装目录\DotNet\
├── E3DCopilot.Addin.dll           # Addin 入口
├── E3DCopilot.Core.dll            # 核心库
├── E3DCopilot.Tools.dll           # 工具库
├── E3DCopilot.UI.dll              # UI 库
├── Newtonsoft.Json.dll            # NuGet 依赖
├── Microsoft.Data.Sqlite.dll      # 记忆系统依赖
└── ...
```

### 步骤 3：配置 PML 索引（可选）

如使用 PML 辅助函数，复制到 PMLLIB 目录：

```
E3D安装目录\PMLLIB\site\e3dcopilot\
├── init.pmlfnc
└── *.pmlfnc
```

### 步骤 4：重启 E3D

E3D 启动时自动扫描 DotNet 目录加载插件。

### 步骤 5：验证

1. 在 E3D 中检查是否出现 "E小智" 工具栏按钮
2. 点击按钮，确认 Copilot 窗口弹出
3. 输入测试指令，验证功能正常

## 七、常见问题

### Q: 插件没有加载？

A: 检查以下几点：
1. 配置文件路径是否正确
2. DLL 是否在 E3D 可访问的路径
3. 依赖的 E3D SDK 版本是否匹配
4. E3D 日志中是否有错误信息

### Q: 窗口没有显示？

A: 检查：
1. `WindowManager.Instance.CreateDockedWindow()` 是否成功调用
2. `UserControl` 是否正确初始化
3. E3D 窗口布局是否隐藏了该窗口

### Q: 权限不足？

A: 确保 E3D 用户有以下权限：
- 加载 Addin 的权限
- 创建 DockedWindow 的权限
- 执行 PML 命令的权限
- 数据库读写权限

## 八、调试技巧

### 使用 E3D 日志

```csharp
// Messaging 静态日志类不存在，用 PML 命令输出到 E3D 消息窗
Command cmd;
cmd = Aveva.Pdms.Utilities.CommandLine.Command.CreateCommand("$p [INFO] 信息消息");
cmd.RunInPdms();
cmd = Aveva.Pdms.Utilities.CommandLine.Command.CreateCommand("$p [WARN] 警告消息");
cmd.RunInPdms();
cmd = Aveva.Pdms.Utilities.CommandLine.Command.CreateCommand("$p [ERR] 错误消息");
cmd.RunInPdms();
// 或 .NET 原生（写事件日志 / Debug 输出）
System.Diagnostics.EventLog.WriteEntry("E3DCopilot", "消息内容",
    System.Diagnostics.EventLogEntryType.Information);
```

### 使用 Windows 事件日志

```csharp
// 在 Windows 事件查看器中查看
EventLog.WriteEntry("E3DCopilot", "启动成功", EventLogEntryType.Information);
```

### 附加调试器

1. 在 Visual Studio 中打开项目
2. 设置 E3D 为启动程序
3. 附加到 E3D 进程
4. 设置断点进行调试
