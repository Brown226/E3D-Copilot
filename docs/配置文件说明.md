# E小智 配置文件说明

## 文件位置

```
%LOCALAPPDATA%\E3DCopilot\config.json
```

完整路径示例：
```
C:\Users\<你的用户名>\AppData\Local\E3DCopilot\config.json
```

> 可通过设置环境变量 `E3DCOPILOT_DATA` 自定义路径，例如：
> `set E3DCOPILOT_DATA=D:\MyE3DData`

---

## 首次启动

首次启动时程序会自动创建 `config.json`，包含默认配置。如需修改，直接编辑该文件，**重启 E3D 生效**。

---

## 关键字段说明

### Ui（界面配置）

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `Ui.Version` | string | `"2.0.0"` | 版本号，显示在设置面板 → 介绍 Tab + 左侧底栏 |
| `Ui.AboutUrl` | string | `""` | 在线说明书链接，为空则不显示"帮助文档"按钮 |
| `Ui.Theme` | string | `"system"` | 主题：`light` / `dark` / `system` |
| `Ui.FontSize` | int | `16` | 默认字号（px）：14=小，16=中，18=大，20=特大 |
| `Ui.DefaultMode` | string | `"act"` | 默认模式：`act`（直接执行）/ `plan`（先规划再执行） |
| `Ui.MaxSteps` | int | `20` | Agent 执行轮数上限（0 = 不限） |
| `Ui.FontFamily` | string | `"default"` | 字体：`default`（系统字体）/ `mono`（等宽字体） |
| `Ui.Notifications` | bool | `true` | 桌面通知开关 |
| `Ui.SoundEnabled` | bool | `false` | 提示音开关 |

### Providers（模型供应商）

每个 Provider 配置：

| 字段 | 类型 | 说明 |
|------|------|------|
| `Name` | string | 供应商唯一标识（如 `openai`、`deepseek`） |
| `Kind` | string | 接入协议（目前仅支持 `openai`） |
| `BaseUrl` | string | API 地址（如 `https://api.openai.com/v1`） |
| `ApiKey` | string | API 密钥 |
| `Models` | string[] | 可用模型列表 |
| `DefaultModel` | string | 默认模型名 |
| `ContextWindow` | int | 上下文窗口大小（0 = 模型默认值） |
| `VisionModels` | string[] | 支持图片输入的模型列表 |

### Iso（ISO出图配置）🆕

ISO出图配置采用**双层架构**：

#### 管理员级别配置（全局 config.json）

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `Iso.DefaultProjectId` | string | `"1907"` | 默认项目编号：`1907` / `1916` / `2016` / `2026` |
| `Iso.DefaultTemplateType` | string | `"standard"` | 默认模板类型：`standard` / `detailed` / `simplified` |
| `Iso.IncludeMaterialList` | bool | `true` | 是否包含材料清单 |
| `Iso.AutoCadTimeoutSeconds` | int | `60` | AutoCAD启动超时时间（秒） |
| `Iso.AutoStartAutoCad` | bool | `true` | 是否自动启动AutoCAD |

#### 用户级别配置（用户 user.json）

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `Iso.AutoCadPath` | string | `""` | AutoCAD可执行文件路径（每台电脑不同） |
| `Iso.DefaultOutputDir` | string | `""` | 默认输出目录（用户可自定义） |

**合并规则**：用户配置 > 全局配置 > 代码默认值

**AutoCAD路径配置方式：**

1. **自动检测**：使用 `manage_autocad` 工具的 `detect_path` 操作，从注册表自动检测
2. **手动配置**：使用 `manage_autocad` 工具的 `save_path` 操作保存路径（保存到用户配置）
3. **直接编辑**：编辑 `user.json` 中的 `Iso.AutoCadPath` 字段

```json
// 全局配置 config.json（管理员维护）
{
  "Iso": {
    "DefaultProjectId": "1907",
    "DefaultTemplateType": "standard",
    "IncludeMaterialList": true,
    "AutoCadTimeoutSeconds": 60,
    "AutoStartAutoCad": true
  }
}

// 用户配置 user.json（用户维护）
{
  "Iso": {
    "AutoCadPath": "C:\\Program Files\\Autodesk\\AutoCAD 2024\\acad.exe",
    "DefaultOutputDir": "D:\\ISO出图结果"
  }
}
```

### Safety（安全配置）

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `Safety.AutoApproveTools` | bool | `false` | 自动批准工具调用 |
| `Safety.AutoApproveEdits` | bool | `false` | 自动批准文件编辑 |

## 修改方式

### 方式一：直接编辑文件（推荐）

1. 关闭 E3D
2. 用文本编辑器打开 `config.json`
3. 修改对应字段
4. 重新启动 E3D

### 方式二：通过设置面板

大部分设置可通过 E3D 中的 **设置面板** 实时修改（无需重启），包括：
- 主题、字号、字体
- 模型参数（Temperature、MaxTokens）
- Agent 执行轮数上限
- 自动审批开关

> 注意：`Version` 和 `AboutUrl` 目前只能通过编辑 config.json 修改。

### 方式三：通过AI工具（ISO配置）

ISO出图相关配置可通过AI工具自动管理：

```
用户：检测AutoCAD安装路径
AI：调用 manage_autocad detect_path → 自动检测并保存

用户：设置默认项目编号为1916
AI：直接修改 config.json 的 Iso.DefaultProjectId 字段
```
