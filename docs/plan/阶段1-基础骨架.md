# Phase 1: 基础骨架 + MVP

> ⏱ **耗时**：6 天（D1 下午 ~ D6）  
> 📅 **前置**：[Phase 0](阶段0-环境准备.md)  
> 🎯 **目标**：在 E3D 中实现 Natural Language → AI → Tool → PML → Result 的完整闭环  

---

## 一、范围限定

### ✅ 本阶段做

| 子阶段 | 内容 |
|--------|------|
| **1a** | Addin 入口 + DockedWindow + PML Engine + AgentLoop 最小闭环 |
| **1b** | 7 个基础 Tool + CsharpBridge + System Prompt |
| **1c** | 错误处理 + 日志 + 配置 + Plan Mode |
| **1d** | 部署脚本 + 完整验证 |

### ❌ 本阶段不做

- 不实现 46 个工具中的 39 个（只做 7 个基础）
- 不实现长期记忆
- 不实现 Session 持久化
- 不实现 Skill 系统
- 不实现 PML 脚本预览（Phase 2 做）
- 不实现上下文压缩（Phase 3 做）

## 七、风险

| 风险 | 影响 | 缓解 |
|------|------|------|
| vLLM 首次启动慢 | AgentLoop 不通 | 先用手动模拟回复测试 UI |
| E3D API 签名仍有幻觉 | 工具执行失败 | 调用前对照 API核实报告.md 核实 |
| WebView2 线程安全 | Bridge.SendToFrontend 跨线程调用异常 | Bridge 内部必须用 Control.Invoke 封送到 UI 线程 |
| config.json 加载失败 | vLLM 地址不可用 | 硬编码默认值 + 失败回退 |
