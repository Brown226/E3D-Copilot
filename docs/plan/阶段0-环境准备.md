## 四、依赖与风险

| 风险 | 缓解 |
|------|------|
| E3D 安装路径不确定 | 用环境变量 `%E3DROOT%` 或配置文件中指定 |
| .NET 4.8 SDK 未安装 | 确认开发机已安装 .NET Framework 4.8 SDK |
| NuGet 源不可用 | 内网环境需配置本地 NuGet 源 |
